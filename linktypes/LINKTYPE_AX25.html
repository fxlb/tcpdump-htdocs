LINKTYPE_AX25

Description

AX.25 frames, as specified by section 3 "Frame Structure" of AX.25 Link Access Protocol for Amateur Packet Radio, Version 2.2, Revision: July 1998. The frames include the address and control fields, but does not include the FCS. The data in the frame is not bit-stuffed, so it includes no flag fields.
