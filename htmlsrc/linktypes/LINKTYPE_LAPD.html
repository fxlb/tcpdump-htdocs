LINKTYPE_LAPD

Description

Link Access Procedures on the D Channel (LAPD) frames, as specified by section 2 "Frame structure for peer-to-peer communication" of ITU-T Recommendation Q.921. The flags at the beginning and end of the frame are not present. If the FCS is present for a frame in a capture file format that supports indicating the presence of an FCS, the file must indicate this for the frame.
